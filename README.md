# SurveyButler
A full-stack (MERN) survey management SaaS app.

[View SurveyButler](https://surveybutler.herokuapp.com/)

The app's stack includes:
- React + Redux
  - React Router
  - Redux Thunk
  - Redux Forms
- Node + Express
  - Passport.js
    - Google OAuth
- SendGrid (Email Service Provider)
  - Email Click Tracking (For Surveys)
- Stripe Payments
- MongoDB
